⚙️ Architecture

Built on AWS

Every layer is isolated, encrypted, and governed by least-privilege IAM — following AWS security best practices end-to-end.

01
🌐
Public Tier
The Application Load Balancer is the only internet-facing resource. It accepts HTTP on port 80 and forwards traffic to the private app tier. No EC2 or RDS is ever exposed.
ALBNAT GatewayPublic Subnetsalb-sg
02
🖥️
App Tier
Node.js runs on EC2 inside a private subnet with no public IP. Access is exclusively via SSM Session Manager — no SSH, no port 22. DB credentials fetched from Secrets Manager on startup.
EC2Node.jsSSMSecrets Manager
03
🐘
Data Tier
RDS PostgreSQL in a fully isolated private subnet — zero internet path. Encrypted at rest with a Customer Managed KMS Key. Only the app tier can reach port 5432 via security group chaining.
RDS PostgreSQLKMS CMKAES-256rds-sg
