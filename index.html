Architecture

How It
Works

01
Public Tier
An Application Load Balancer sits at the edge, accepting HTTP traffic on port 80 from the internet. No direct access to EC2 or RDS is possible.
ALB · Internet-facing
02
App Tier
A Node.js server runs on EC2 inside a private subnet. No public IP. Accessed only via ALB. Fetches database credentials from Secrets Manager on startup.
EC2 · Node.js · SSM
03
Data Tier
RDS PostgreSQL runs in a private subnet with no internet path. Encrypted at rest with a Customer Managed KMS Key. Only reachable from the app tier.
RDS · KMS · Encrypted
